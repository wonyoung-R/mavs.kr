# Instagram Captions (수동 드롭)

@mavs.kr 인스타그램에 올린 캡션 텍스트를 한 파일당 하나씩 `.md`로 드롭하세요.

## 파일명 규칙
```
YYYY-MM-DD_short-slug.md
```

예:
- `2026-01-15_flagg-debut.md`
- `2026-02-20_kidd-rotation.md`

## 파일 포맷
```markdown
---
posted_at: 2026-01-15
url: https://instagram.com/p/Cxxxx/
type: card_news | photo | reels
---

[캡션 본문 전체]
```

이 폴더의 파일들은 자동 글 생성 시 VOICE_SPEC.md 보완 학습 재료로 사용됩니다.
지금은 비어있어도 됩니다 — 시간 날 때마다 채워주세요.
