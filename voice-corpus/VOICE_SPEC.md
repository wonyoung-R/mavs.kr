## 1. 톤 특성

### 1.1 문장 길이
- 평균 30~80자
- 한 단락 2~4문장
- 인터뷰 인용 시 따옴표 + 인용자 명시 + 본인 코멘트 별도 단락 분리

### 1.2 격식 수준
- **캐주얼-공적 혼용**. 어미 자유:
  - 서술형: "한다", "이다", "뿐이다"
  - 정중형: "~입니다", "~습니다" — 인용/공식 발언 옮길 때
- 어미를 일관시키지 않고 글 안에서 자연스럽게 섞음
- 블로그/SNS 톤 X. 한국 스포츠 기사 톤 ✅

### 1.3 감정 표현
- **사람 작성 시:** "에휴", "안타깝게도", "불쌍한", "ㅠ", "🥲" 자연스럽게 등장
- **자동 글에서는 절대 금지:** 이모지 0개, 감정 부사 자제, 객관 관찰형 유지

## 2. 자주 쓰는 어휘·표현

### 2.1 등장 패턴
- "안타깝게도 ~ㅠ"
- "에휴, ~"
- "꼬일대로 꼬여버린"
- "~로 부터 까이는거 보면"
- "정말 ~ 좋은 경험"
- "참고로", "사실은", "솔직히"

### 2.2 인용 패턴
```
[인용자 이름]은 [공식 직함]은(는) [발언 요지]에 대해 이렇게 말했습니다.
"[실제 발언]"
라고 [인용자]는 말했습니다.
```

### 2.3 출처 명기
- 외신 출처 링크 본문 끝에 첨부
- 캡쳐 이미지 별도 단락에 삽입
- mania.kr, ESPN, MMB 등 명확하게

## 4. 피해야 하는 표현

### 4.1 자동 글에서 절대 금지 (banned-patterns.ts에 정규식 등록됨)
- 처방형: "~해야 한다", "~할 필요가 있다"
- 단정 미래: "~할 것이다", "~될 것이다"
- 비방어: 실패작, 망했다, 한심, 무능, 쓰레기, 개판
- 인사 결정: 경질, 해고, "트레이드해야"
- 과장: 충격적, 혁명적, 경악, 화제, 폭로, 미친 활약, 대박
- 메타 언급: 조회수, 좋아요, 댓글 수, 공유 수, 트래픽
- 모든 이모지

### 4.2 사람 작성 글에서도 권장하지 않는 것
- 출처에 없는 사실 단정
- 익명 소스 단독 인용 (확인된 발화자만)

## 6. 시각 박스 작성 템플릿 (Phase 1과 동기화)

### LOW (routine recap, player performance, emotional moment)
- 2~4문장, 80~200자
- 관찰형 어미: "~한 모습이 보였다", "~로 느껴진다", "~는 흐름이다"
- 한국 팬덤 정서 1개 자연스럽게 연결 가능

### MEDIUM (transaction, injury)
- 2~3문장, 80~150자
- "매버릭스 팬덤이 주목하는 변수다" 형태
- 의견/전망 금지, 사실 진술만

### HIGH (controversy, speculation)
- 1~2문장, 50~100자
- 맥락 진술만, 입장 없음
- "신중하게 지켜보는 흐름" 류
